#JPACrudProject

###Description
* Dynamic web application designed as recreate the popular "Street Fighter" video game. 

###TECHNOLOGIES USED 
* Spring Tool Suite
* HTML
* Inheritance 
* MAMP
* MYSQL
* MYSQL Workbench
* JSP
* TOMCAT
* Spring Boot
* JPA


###HOW TO RUN 
* From the home page, select from a menu of options to create, search, update, and delete different players.
